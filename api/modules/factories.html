    Module factories

    Factory Pattern — Agent creation and registration

    Provides factory pattern implementation for creating agent instances. Supports 6+ agent types: Planner, Roadmapper, Executor, Phase Researcher, Project Researcher, and Verifier.

    Example

    import { AgentFactoryRegistry, EzPlannerAgent } from '@howlil/ez-agents';

    // Register agent
    AgentFactoryRegistry.register('planner', EzPlannerAgent);

    // Create agent instance
    const planner = AgentFactoryRegistry.create('planner', {
      model: 'qwen',
      temperature: 0.7
    });

    // Execute task
    const result = await planner.execute({
      task: 'Create project roadmap',
      context: [...]
    });
