- bin/lib/strategies/RankByRelevanceStrategy.ts

</read_first>

<action>
Create `bin/lib/strategies/StrategyFactory.ts`:

1. Import all strategy classes
2. Import `CompressionStrategy` interface
3. Export `createStrategy(name: string, options?: any): CompressionStrategy` function:
   ```typescript
   export function createStrategy(name: string, options?: any): CompressionStrategy {
     switch (name) {
       case 'summarize':
         return new SummarizeStrategy(options?.modelName);
       case 'truncate':
         return new TruncateStrategy(options?.maxTokens);
       case 'rank-by-relevance':
         return new RankByRelevanceStrategy(options?.scorer);
       case 'hybrid':
         return new HybridStrategy(options);
       default:
         throw new Error(`Unknown strategy: ${name}`);
     }
   }
   ```

4. Export `getAvailableStrategies(): string[]` function
</action>
