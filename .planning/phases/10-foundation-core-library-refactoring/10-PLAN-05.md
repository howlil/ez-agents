<action>
Create `bin/lib/adapters/AdapterFactory.ts`:

1. Import all adapter classes
2. Import `ModelProviderAdapter` interface
3. Export `createModelAdapter(name: string, options: AdapterOptions): ModelProviderAdapter` function:
   ```typescript
   export function createModelAdapter(name: string, options: AdapterOptions): ModelProviderAdapter {
     switch (name) {
       case 'claude':
         return new ClaudeAdapter(options.apiKey, options.modelName);
       case 'openai':
         return new OpenAIAdapter(options.apiKey, options.modelName);
       case 'kimi':
         return new KimiAdapter(options.apiKey, options.modelName);
       case 'qwen':
         return new QwenAdapter(options.apiKey, options.modelName);
       default:
         throw new Error(`Unknown model provider: ${name}`);
     }
   }
   ```

4. Export `getAvailableAdapters(): string[]` function
</action>
