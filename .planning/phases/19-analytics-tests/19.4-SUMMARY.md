## Methods Implemented

1. **defineCohort(cohort)**
   - Creates cohort with name, startDate, endDate
   - Persists to `.planning/cohorts.json`

2. **addUserToCohort(userId, signupDate)**
   - Adds user to matching cohort
   - Updates memberships map

3. **calculateRetention(cohortName, options)**
   - Calculates weekly retention (4 weeks default)
   - Returns RetentionResult with periods array

4. **compareCohorts(cohortNames)**
   - Compares multiple cohorts
   - Returns size and retention for each

## Data Structure

```json
{
  "cohorts": [{ "name": "jan_2026", "startDate": "...", "endDate": "..." }],
  "memberships": { "jan_2026": ["user1", "user2"] },
  "activities": { "user1": [{ "userId": "user1", "timestamp": "..." }] }
}
```
